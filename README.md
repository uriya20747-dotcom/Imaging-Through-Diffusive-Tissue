# Imaging Through Diffusive Tissue

Final Project - Electrical and Electronics Engineering

## Overview

This project explores image reconstruction through a diffusive medium using an Encoder-Decoder neural network.

The system combines:

- Raspberry Pi
- Waveshare Display
- Basler Camera
- UART Communication
- Image Processing Pipeline
- Deep Learning (Encoder-Decoder Network)

## Project Structure

- data_preparation
- simulation
- neural_network
- hardware_system
